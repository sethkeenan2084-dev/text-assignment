# text-assignment
This project focuses on using natural language processing (NLP) to analyze and compare multiple text files in order to better understand their content and writing style. The main goal is to take unstructured text and turn it into something more meaningful by applying techniques like tokenization, stemming, lemmatization, named entity recognition, and trigram analysis. Instead of using a complex class structure, the program is designed with simple, reusable functions that each handle a specific task, such as loading files, cleaning the text, counting word frequencies, identifying named entities, and generating n-grams. Key attributes include the text variables for each file, a set of stopwords, and tools like the stemmer, lemmatizer, and spaCy model. While the methods work well for identifying patterns and themes, there are some limitations, like not fully understanding context, possible inaccuracies in entity recognition, and the fact that trigram analysis alone cannot definitively prove authorship. Overall, the project shows how NLP can be used in a practical way to extract insights from text and compare documents more effectively.
